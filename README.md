# 🧮 Калькулятор на Go

Простой и элегантный калькулятор с веб-интерфейсом, написанный на Go. Поддерживает базовые арифметические операции.

## ✨ Особенности

- ➕ Сложение, вычитание, умножение, деление
- 🎨 Современный адаптивный дизайн
- ⌨️ Поддержка клавиатурного ввода
- 📱 Работает на всех устройствах
- ⚡ Быстрый и легковесный
- 🔒 Безопасная обработка ошибок (деление на ноль и др.)

## 🚀 Быстрый старт

### Требования
- Go 1.21 или выше
